Installation Guide

Since CrossDrop is in active development, you need to install it manually in Developer Mode.

1

Download the Extension

Download the latest crossdrop-extension.zip from our home page or GitHub Releases.

2

Extract Files

Unzip the downloaded file to a folder on your computer.

3

Open Extensions Page

Open Google Chrome (or Edge) and navigate to chrome://extensions.

4

Enable Developer Mode

Toggle the Developer mode switch in the top right corner of the page.

5

Load Unpacked

Click the Load unpacked button that appears. Select the folder where you extracted the extension.
